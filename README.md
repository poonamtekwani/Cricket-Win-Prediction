# 🏏 Cricket Win Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting the performance and match outcomes of the Indian cricket team using machine learning techniques. It leverages historical match data to generate insights and support data-driven decision making.

---

## 🎯 Objectives
- Predict match outcomes (Win/Loss)  
- Identify key factors influencing performance  
- Provide strategic recommendations  

---

## 📊 Dataset
- ~2600+ match records  
- Features include player performance, team composition, match format, and conditions  

---

## ⚙️ Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest ⭐ (Best Model)  

---

## 📈 Model Performance
- Random Forest Accuracy: **~94%**  
- High precision, recall, and ROC-AUC score  

---

## 🔍 Key Insights
- Player performance significantly impacts match results  
- Team composition (all-rounders, bowlers) is crucial  
- Match conditions and format influence outcomes  

---

## 🧠 Techniques Used
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- SMOTE (Handling class imbalance)  
- Feature Engineering  
- Model Evaluation  

---

## 🛠️ Tools Used
Python | Pandas | NumPy | Scikit-learn | Matplotlib | Seaborn  

---

## 📌 Conclusion
Random Forest model provided the best performance and helped generate actionable insights for improving team strategies.
