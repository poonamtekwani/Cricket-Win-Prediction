# 🏏 Cricket Win Prediction using Machine Learning

## 📌 Project Overview
Developed a machine learning model to predict match outcomes of the Indian cricket team using historical match data and provide data-driven strategic insights.

---

## 🎯 Objectives
- Predict match outcomes (Win/Loss)  
- Identify key factors affecting performance  
- Provide actionable recommendations for improvement  

---

## 📊 Dataset
- 2600+ match records after preprocessing  
- Features include player performance, team composition, match format, and conditions  

---

## ⚙️ Approach
- Data Cleaning & Missing Value Treatment  
- Exploratory Data Analysis (EDA)  
- Feature Engineering & Encoding  
- Outlier Treatment using IQR  
- Class imbalance handled using SMOTE  

---

## 🤖 Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest ⭐ (Best Model)  

---

## 📈 Model Performance
- Random Forest Accuracy: **94.15%**  
- Precision: **94.53%**  
- Recall: **94.15%**  
- ROC-AUC: **95.42%**  

---

## 🔍 Key Insights
- Player performance significantly impacts match results  
- Team composition (all-rounders, bowlers) plays a crucial role  
- Match format and conditions influence outcomes  
- Opponent strength affects performance trends  

---

## 💡 Business Impact
- Helps optimize team selection strategies  
- Supports match-specific decision-making  
- Improves chances of winning using data-driven insights  

---

## 🛠️ Tools & Technologies
Python | Pandas | NumPy | Scikit-learn | Matplotlib | Seaborn  

---

## 📌 Conclusion
Random Forest model delivered the best performance and provided reliable predictions along with actionable insights to enhance team strategy and performance.
