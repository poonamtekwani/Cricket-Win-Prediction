# 🏏 Cricket Win Prediction using Machine Learning

## 📌 Project Overview
This project predicts match outcomes of the Indian cricket team using machine learning models.

---

## 🎯 Objectives
- Predict match outcomes (Win/Loss)
- Identify key performance factors
- Provide data-driven strategies

---

## ⚙️ Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest ⭐  

---

## 📈 Results
- Achieved ~94% accuracy using Random Forest  
- Identified key features affecting match outcomes  

---

## 🛠️ Tools Used
Python | Pandas | NumPy | Scikit-learn | Matplotlib  

---

## 📌 Conclusion
Random Forest performed best and provided accurate predictions and useful insights.
